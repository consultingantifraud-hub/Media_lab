# Инструкция по расширению диска C:

## Текущая ситуация:

После удаления раздела Ubuntu у вас есть нераспределенное пространство (~47 ГБ), но оно находится **после раздела восстановления**, а диск C: находится **перед** разделом восстановления.

## Варианты решения:

### Вариант 1: Расширение через Disk Management (рекомендуется)

1. Откройте **Disk Management** (`diskmgmt.msc`)
2. Найдите диск **C:** (189.38 GB)
3. **Правой кнопкой** по диску C: → **"Расширить том"** (Extend Volume)
4. Если опция доступна:
   - Следуйте инструкциям мастера
   - Выберите максимальный размер или нужный размер
   - Завершите операцию
5. Если опция **недоступна** (серым цветом):
   - Это означает, что между C: и нераспределенным пространством есть раздел восстановления
   - Переходите к Варианту 2

### Вариант 2: Если расширение недоступно

Windows не позволяет расширять диск C: через раздел восстановления. В этом случае у вас есть два варианта:

#### Вариант 2A: Оставить как отдельный раздел (безопасно)

1. В Disk Management найдите нераспределенное пространство
2. **Правой кнопкой** → **"Создать простой том"** (New Simple Volume)
3. Назначьте букву диска (например, D:)
4. Используйте это пространство как отдельный диск для хранения данных

#### Вариант 2B: Переместить раздел восстановления (продвинутый)

⚠️ **ВНИМАНИЕ:** Это более сложная операция, требующая специальных инструментов.

1. Используйте инструменты вроде **AOMEI Partition Assistant** или **MiniTool Partition Wizard**
2. Переместите раздел восстановления в конец диска
3. Затем расширьте диск C:

### Вариант 3: Через PowerShell скрипт

Попробуйте запустить скрипт (может не сработать из-за разделов между C: и нераспределенным пространством):

```powershell
# Запустите от имени администратора
cd C:\MAIN\Bots\Media_lab\docker
.\extend-c-drive.ps1
```

## Рекомендация:

**Используйте Вариант 1** (Disk Management). Если расширение недоступно, используйте **Вариант 2A** - создайте отдельный диск для хранения данных. Это безопасно и не требует сложных операций.

---

**Примечание:** Раздел восстановления важен для восстановления системы Windows, поэтому его удаление не рекомендуется.
