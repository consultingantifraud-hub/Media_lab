# Решение: Расширение диска C: когда кнопка неактивна

## Проблема:
Кнопка "Расширить том" неактивна, потому что между диском C: и нераспределенным пространством находится **раздел восстановления**. Windows не позволяет расширять диск через другие разделы.

## Решения (от простого к сложному):

### ✅ Решение 1: Создать отдельный диск (РЕКОМЕНДУЕТСЯ - самое безопасное)

**Преимущества:**
- ✅ Полностью безопасно
- ✅ Не требует сложных операций
- ✅ Не рискуете повредить систему
- ✅ Можно использовать сразу

**Шаги:**
1. Откройте **Disk Management** (`diskmgmt.msc`)
2. Найдите **нераспределенное пространство** (~47 GB, черное)
3. **Правой кнопкой** → **"Создать простой том"** (New Simple Volume)
4. Назначьте букву диска (например, **D:**)
5. Отформатируйте в NTFS
6. Готово! Теперь у вас есть диск D: на 47 GB

**Использование:**
- Храните на нем данные, файлы, проекты
- Docker можно настроить использовать этот диск
- Или просто используйте для хранения

---

### ⚠️ Решение 2: Переместить раздел восстановления (продвинутое)

**Требуется:** Специализированные инструменты

#### Вариант 2A: AOMEI Partition Assistant (Бесплатно)

1. **Скачайте:** https://www.diskpart.com/free-partition-manager.html
2. **Установите** и запустите от имени администратора
3. **Правой кнопкой** по разделу восстановления → **"Move/Resize Partition"**
4. Переместите раздел в **конец диска**
5. Примените изменения (Apply)
6. После перемещения расширьте диск C: через Disk Management

#### Вариант 2B: MiniTool Partition Wizard (Бесплатно)

1. **Скачайте:** https://www.partitionwizard.com/free-partition-manager.html
2. Аналогичные шаги как в варианте 2A

**⚠️ ВНИМАНИЕ:**
- Создайте резервную копию важных данных перед операцией
- Операция может занять время
- Не прерывайте процесс

---

### ❌ Решение 3: Удалить раздел восстановления (НЕ РЕКОМЕНДУЕТСЯ)

**Не рекомендуется**, потому что:
- Раздел восстановления нужен для восстановления Windows
- Без него сложнее восстановить систему при проблемах
- Можно удалить, но лучше оставить

---

## Рекомендация:

**Используйте Решение 1** - создайте отдельный диск D: из нераспределенного пространства. Это:
- ✅ Безопасно
- ✅ Быстро
- ✅ Не требует сторонних инструментов
- ✅ Не рискуете повредить систему

Если вам **обязательно нужно** расширить именно диск C:, используйте **Решение 2** с инструментом AOMEI Partition Assistant.

---

## После создания диска D:

Вы можете:
- Использовать его для хранения данных
- Настроить Docker использовать этот диск (изменив путь в настройках)
- Хранить там проекты, файлы, медиа
