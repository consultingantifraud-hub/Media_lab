# Инструкция по загрузке файлов на сервер

## Вариант 1: Через новое окно PowerShell (рекомендуется)

1. Откройте **новое окно PowerShell** на вашем локальном компьютере
2. Выполните команды:

```powershell
cd C:\MAIN\Bots\Media_lab
scp -r * reg-ru-neurostudio:/opt/media-lab/
```

Введите пароль когда запросит.

## Вариант 2: Через терминал сервера (если есть Git репозиторий)

Если ваш проект в Git репозитории, на сервере выполните:

```bash
cd /opt/media-lab
git clone <ваш-репозиторий> .
```

## Вариант 3: Через архив (вручную)

1. На локальном ПК создайте архив проекта
2. Загрузите через WinSCP или FileZilla
3. На сервере распакуйте: `unzip project.zip -d /opt/media-lab`
