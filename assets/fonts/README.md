# Шрифты для Stylish text

Для работы функции Stylish text необходимо добавить следующие шрифты:

## Требуемые шрифты

1. **Inter-Bold.ttf** - для основного текста (поддержка кириллицы)
   - Скачать: https://fonts.google.com/specimen/Inter
   - Или использовать другой шрифт с поддержкой кириллицы

2. **NotoColorEmoji.ttf** - для цветных эмодзи
   - Скачать: https://github.com/googlefonts/noto-emoji
   - Или использовать системный шрифт эмодзи

## Установка

Поместите файлы шрифтов в эту директорию:
- `assets/fonts/Inter-Bold.ttf`
- `assets/fonts/NotoColorEmoji.ttf`

Если шрифты не найдены, система будет использовать fallback шрифты (могут не поддерживать эмодзи).
